Ultima Online Protocol :: Destroy Static
Packet # Packet Name Length Client Server Obsolete
0x61 Destroy Static 0x0009 X
Data Type Description
byte 0x61 The packet number.
ushort x The x-axis position where the tile is.
ushort y The y-axis position where the tile is.
short z The z-axis position where the tile is.
ushort tile_num The tile number.

Packet Description:
This god client packet is sent to destroy a static object.

To destroy a static, hold shift and select the item or items and then press the delete key.
