Ultima Online Protocol :: IGR Logout
Packet # Packet Name Length Client Server Obsolete
0xD1 IGR Logout 0x0002 X
Data Type Description
byte 0xD1 The packet number.
bool status If true, the client is requesting to log off or the server is telling it okay.

Packet Description:
IGR clients send this packet to log off, and will wait until it receives this packet from the server before logging off.
