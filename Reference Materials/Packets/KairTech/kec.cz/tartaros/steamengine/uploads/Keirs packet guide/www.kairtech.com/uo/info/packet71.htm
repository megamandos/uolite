Data Type Description
byte 0x71 The packet number.
ushort Size The size of the packet.
byte Command The command being sent.

  • 0x00 - Display Bulletin Board
  • 0x01 - Message Summary
  • 0x02 - Message Data
  • 0x03 - Request Message
  • 0x04 - Request Message Summary
  • 0x05 - Post Message
  • 0x06 - Delete Message

Packet Description:
This packet is used to perform various bulletin board functions.
