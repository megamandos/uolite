Data Type Description
byte 0x1C The packet number.
ushort Size The size of the packet.
uint Serial The serial of the object displaying the message.

0xFFFFFFFF is used for system messages.
ushort Body The body number of the object displaying the message.

0xFFFF is used for system messages.
byte Type The type of message to send.

  • 0x00 - Normal
  • 0x01 - Broadcast
  • 0x02 - Emote
  • 0x06 - System
  • 0x07 - Message
  • 0x08 - Whisper
  • 0x09 - Yell
ushort Hue The color of the text.
ushort Font The text's font number. Normal message font is 0x0003.
char[30] Name The name of the speaker. "System" is used for system messages.
char[] Text The text to display.
byte 0x00 Null terminator for Text.

Packet Description:
This is sent to display character names, system messages, object messages, etc.
